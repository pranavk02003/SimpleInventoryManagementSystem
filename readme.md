 Simple Inventory Management System



The purpose of this project is to perform basic CRUD operations on product data such as:
- Viewing products
- Adding a new product
- Deleting a product
- Updating a product

 How This Project Was Done

- Created an Express server using Node.js
- Organized the project using separate folders for routes, controllers, and data
- Stored product data in a JavaScript file
- Implemented CRUD operations using REST API endpoints
- Tested all APIs using Postman
- Used Nodemon for automatic server restart during development
